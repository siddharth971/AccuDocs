# Security Policy

## Supported Versions

We currently support and provide security updates for the following versions:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of AccuDocs seriously. If you discover a security vulnerability, please do NOT create a public issue. Instead, please report it privately.

### How to report

Please report vulnerabilities via email to **support@accudocs.example.com**.

Please include:

- A clear description of the issue.
- Detailed steps to reproduce.
- Any potential impact or proof of concept.

### Our Commitment

- We will acknowledge your report within 48 hours.
- We will keep you updated on the progress of the fix.
- We will release a security patch as soon as possible.

Thank you for helping keep AccuDocs secure!
